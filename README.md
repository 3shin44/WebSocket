# WebSocket practice

## Introduction 說明
Create local WebSocket server and client.  
建立本地端WebSocket伺服器與用戶端

## NPM install
```
npm install
```

### Launch WS server 伺服器啟動
```
node ws_server.js
```

### Launch WS client 用戶端啟動
```
npm run server
```
